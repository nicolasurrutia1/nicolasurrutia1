Buttons


Regular

Add classes .btn and .btn-{red|orange|tangerine|yellow|rose|black|dark-grey|grey|light-grey|lighter-grey|white} to the <a> tag

.btn-red .btn-orange .btn-tangerine .btn-yellow .btn-rose
.btn-black .btn-dark-grey .btn-grey .btn-light-grey .btn-lighter-grey .btn-white

Large

Add classes .btn .large and .btn-{red|orange|tangerine|yellow|rose|black|dark-grey|grey|light-grey|lighter-grey|white} to the <a> tag

.large .btn-red .large .btn-orange .large .btn-tangerine .large .btn-yellow .large .btn-rose
.large .btn-black .large .btn-dark-grey .large .btn-grey .large .btn-light-grey .large .btn-lighter-grey .large .btn-white

Text links


The correct tags to use in a text-link is a tag <a> with the class of the text-link, then inside the tag <a> you must add the tag of the text you want, for example <h3>, <h4>, etc.

Add class .text-link and .{red|orange|tangerine|yellow|rose|black|dark-grey|grey|light-grey|lighter-grey|white}-link to the <a> tag

.text-link red-link .text-link orange-link .text-link tangerine-link .text-link yellow-link .text-link rose-link

.text-link black-link .text-link dark-grey-link .text-link grey-link .text-link light-grey-link .text-link lighter-grey-link
.text-link white-link

Loader

This is a page loader that you can use in pages that require big files to load. just add the HTML Code right after the body.
there are three different themes, so you can add the one that matches your page color scheme!, just change the clasess to: Classic, darkMode & lightMode.
